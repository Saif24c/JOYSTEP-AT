# JOYSTEP AT

## Overview

JOYSTEP AT is an AI-powered Assistive Technology platform designed to help individuals with motor disabilities interact with computers using hand gestures, facial movements, and voice commands.

The system provides a hardware-free alternative to expensive assistive devices by utilizing a standard webcam and microphone.

## Features

### Hand Gesture Control

* Cursor movement using hand tracking
* Left click using thumb gesture
* Right click using pinch gesture
* Scroll mode using peace sign gesture
* Cursor freeze using fist gesture

### Face Control Mode

* Nose-based cursor navigation
* Dynamic Sensitivity Mapping (DSM) calibration
* Wink-based left click
* Long blink right click
* Mouth-open click gesture

### Voice Assistant (JARVIS)

* Wake word activation
* Voice typing and dictation
* Cursor control through voice
* Window management
* Application launching
* Screenshot capture
* Volume and brightness control
* System operations

### Accessibility Focus

* Controller-free interaction
* Webcam-based tracking
* Voice-only operation support
* Designed for users with motor impairments
